1. `x` is one-indexed
2. Assume that all inputs are valid - `x` will never be more than the number of digits in `combo`

```
function x_order(order, x) {
    // YOUR SOLUTION HERE 
}

// Examples
x_order(1234, 1) // 4
x_order(1234, 3) // 2
x_order(12345678, 8) // 1x_combo
```
